Explicit Euler

Given a first order ordinary differential equation:

dydx=f(x,y)

where y=y(x), using forward differences

dydx→y(x+Δx)−y(x)(x+Δx)−x=y(x+Δx)−y(x)Δx=f(x,y)

we get the famous Forward-Euler formula

y(x+Δx)=y(x)+f(x,y)Δx

Using the notation x0+n⋅Δx=xn  and y(xn)=yn we get

yn+1=yn+f(xn,yn)Δx

Implicit Euler

If instead of using forward differences, we use backward differences, we get

dydx→yn−yn−1xn+1−xn=yn−yn−1Δx=f(xn,yn)

from which we get the Implicit-Euler formula

yn=yn−1+f(xn,yn)Δx

or, equivalently

yn+1=yn+f(xn+1,yn+1)Δx

Note that these equations need to be solved for yn+1.

Second order ordinary differential equation

A second order ordinary differential equation of the form

y″=f(x,y,y′),y′=dydx,y″=d2ydx2

can be reduced to two first order ordinary differential equations

uu′==dydxf(x,y,u)

Then these two can be solved simultaneously for the value of the next step by using any method desired.

Using forward Euler:

yn+1un+1==yn+unΔxun+f(xn,yn,un)Δx

And implicit Euler:

yn+1un+1==yn+un+1Δxun+f(xn+1,yn+1,un+1)Δx

If the function f is linear in x, u and u′, then this latter equation set can be solved by solving corresponding the matrix equation.

Example: Harmonic Oscillator

The harmonic oscillator is a second order, linear, ordinary differential equation

y″=−ω2y

which, when solved using explicit and implicit Euler yields

yn+1un+1==yn+unΔxun−ω2ynΔx

and

yn+1un+1==yn+un+1Δxun−ω2yn+1Δx

this latter equation set can be rewritten as

(1ω2Δx−Δx1)(yn+1un+1)=(ynun)

which can be solved, and gives

(yn+1un+1)=11+ω2Δx2(1−ω2ΔxΔx1)(ynun)
