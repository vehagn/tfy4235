Box-Müller algorithm

The algorithm to generate Gaussian numbers is

double x1 = ranf();                  // uniformly distributed on the unit interval
double x2 = ranf();                  // uniformly distributed on the unit interval

double r = sqrt(-2.0*log(x1));
double y1 = r*cos(2*PI*x2);          // gaussian
double y2 = r*sin(2*PI*x2);          // gaussian

Metropolis

Given a probability density function $p(x)$. The Metropolis Monte Carlo algorithm does a random walk, where the random walker's position follows the given $p(x)$.

Algorithm:

  1. Given the current position, $x_i$.
  2. The (gaussian) probability density of being at that position is $p_i = p(x_i)$.
  3. Choose a new temporary position, $x_{t} = x_i + r\Delta x$, where $r\in [-0.5,0.5]$ is an uniformly distributed random number.
  4. Calculate the new probability density, $p_{t} = p(x_{t})$.
  5. If $p_{t}/p_{i} > 1$, that is, the new position is more probable, let $x_{i+1} = x_t$. That is, move to the new position.
  6. If $p_t/p_i \lt 1$, generate a random number, $r \in [0,1]$. If $p_t/p_i \gt r$, move to the new position, $x_{i+1} = x_t$.
  7. Else, $x_{i+1} = x_{i}$.
  8. Go to 1.

$x_{i}$ will now be distributed according to $p(x)$.

Notice: You only need to know the ratio $p_t/p_i$, not the normalization.

Example: (Normal) Gaussian numbers

$$ \frac{p(x_a)}{p(x_b)} = e^{-\frac{1}{2} (x^2_a - x^2_b)} $$

(Auto)correlation function

For a (time) series, $x_i$, the autocorrelation function is, when $\mu = 0$ and $\sigma^2 = 1$:

$$ W(j) = \frac{1}{N-j} \sum_{i=0}^{N-j-1} x_{i} x_{i+1} $$

We only need a finite number of $j$, approximitely 500. (Look at the figures below)
